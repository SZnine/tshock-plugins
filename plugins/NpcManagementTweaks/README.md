# NpcManagementTweaks

面向 `TShock 6.1.0 / Terraria 1.4.5.6` 的可配置世界机制插件。

第一版功能：

- 树木生长速度与高度
- 草药成熟去限制与加速生长
- 旅商保证每日到来并刷新商品
- 渔夫任务每日不限交一次，并可每次交付后刷新任务
- 自定义多线钓鱼（实验功能）

## 配置文件

生成位置：

- `tshock/NpcManagementTweaks.json`

## 管理命令

- `/cwt reload`
- `/cwt status`

## 说明

- 树木功能当前针对普通树苗生效。
- 草药功能通过额外成熟与种植流程提高全图生长速度。
- 旅商功能在每日清晨强制生成，并可重置当日商店。
- 渔夫功能在玩家成功交任务后重置当日限制，并可刷新新任务。
- 多线钓鱼通过额外生成 bobber 实现，属于实验项，建议先在测试服验证。

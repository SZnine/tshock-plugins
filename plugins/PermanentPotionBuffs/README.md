# PermanentPotionBuffs

面向 `TShock 6.1.0 / Terraria 1.4.5.6` 的药水与食物增益插件。

作者：`鱼仔仔面`

## 当前功能

- 所有可识别的增益药水与食物都可参与判定
- 同种药水或食物累计达到 `30` 个时，获得对应常驻增益
- 普通手动使用的非常驻药水，持续时间按原版 `2 倍` 处理
- 普通手动使用的非常驻药水，死亡后尝试继承死亡前剩余时间
- 常驻增益会在玩家可访问容器中持续检测并自动补时

## 统计范围

插件会统计玩家可直接访问的这些位置：

- 身上背包
- 猪猪存钱罐 `Piggy Bank`
- 保险库 `Safe`
- 防御者熔炉 `Defender's Forge`
- 虚空保险库 `Void Vault`

## 当前规则

### 1. 常驻增益

- 同一种药水累计达到 `30` 瓶，启用该药水的常驻 Buff
- 同一种食物累计达到 `30` 个，启用该食物对应的食物 Buff
- 常驻增益会持续刷新，不依赖玩家重新手动使用
- 玩家死亡或复活后，插件会重新补回常驻增益

### 2. 普通药水增强

- 对未达到 `30` 个条件、但玩家手动正常使用的药水生效
- 默认持续时间为原版 `2 倍`
- 插件会记录普通药水的剩余时间
- 玩家死亡后，普通药水会尝试恢复为死亡前记录的剩余时间，而不是重新从满时间开始

## 配置文件

生成位置：

- `tshock/PermanentPotionBuffs.json`

当前关键参数：

- `RequiredStack = 30`
- `PotionDurationMultiplier = 2`

## 管理命令

- `/ppb reload`
- `/ppb status`
- `/ppb status 玩家名`

`status` 可用于查看：

- 当前配置摘要
- 玩家已命中的常驻药水数量
- 当前食物常驻状态
- 普通药水 Buff 的记录状态与剩余时间
